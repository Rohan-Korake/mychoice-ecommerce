## Tech Stack

- **Language:** JavaScript (ES6+)
- **Frontend Framework:** React
- **Build Tool:** Vite
- **Styling:** Tailwind CSS
- **Routing:** React Router DOM
- **Icons:** Lucide React
- **Capabilities:** Progressive Web App (PWA)

---

## Key Features

- **Modern UI:** A clean, responsive design built with **Tailwind CSS**.
- **Fast Loading:** Optimized development and production builds using **Vite**.
- **PWA Support:** Installable on devices for an app-like experience with offline capabilities.
- **Dynamic Routing:** Seamless page transitions powered by **React Router DOM**.
- **WhatsApp Integration:** Quick contact buttons for customer inquiries using `wa.me`.
- **Lucide Icons:** Integrated lightweight and scalable icons for a professional look.
